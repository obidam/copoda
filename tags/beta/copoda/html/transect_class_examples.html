Examples of Transect object creation

We introduced all properties of the Transect class. In the following we give several workflow examples to set up a Transect object with all properties filled. These examples are available in the package and can be called by:

>> demo(transect,EXAMPLE_i); 

Example 1

>> demo(transect,1);

Let's try to mimic a Transect object with OVIDE 2002 informations and datas.

Create an empty default Transect object and fill in basic properties:

T = transect;
T.source = 'The best lab in the world !';
T.creator = 'John Doe';
T.file = 'no_source_file_at_this_point';
T.file_date = datenum(1000,1,1,0,0,0);
T.created = now;

Fill in Cruise informations:

T.cruise_info = cruise_info('NAME','OVIDE 2002','PI_NAME','H. Mercier','PI_ORGANISM','LPO-CNRS','SHIP_NAME','Thalassa','SHIP_WMO_ID','FNFP','DATE',[datenum(2002,6,18,0,0,0) datenum(2002,7,10,0,0,0)],'N_STATION',100);

Fill in axis informations:

T.geo.STATION_NUMBER = [1:100]';
T.geo.STATION_DATE = linspace(datenum(2002,6,18,0,0,0),...
datenum(2002,7,10,0,0,0),100)';
T.geo.LATITUDE = linspace(40.3,59.8,100)';
T.geo.LONGITUDE = linspace(350.5,317.4,100)';
T.geo.POSITIONING_SYSTEM = 'GPS';
T.geo.PRES = meshgrid(1035*9.8*[0:10:5330]/10000,1:100);
T.geo.MAX_PRESSURE = 1035*9.8*5330/10000*ones(100,1);
T.geo.DEPH = meshgrid(0:-10:-5330,1:100);

Fill in datas:

T.data = ...

struct('TEMP',odata('name','TEMP','long_name','Temperature','unit','degC','cont',...
meshgrid(linspace(2,1,100),1:534)'.*meshgrid(1+20*exp(T.geo.DEPH(1,:)/5e2),1:100)));

Example 2

>> demo(transect,2);

Let's again try to mimic a Transect object with OVIDE 2002 informations and datas. This time we will create preliminary structures and fill the Transect properties at the end. We also make use of a virtual variable: the oxygen solubility.

Cruise informations into variable C:

C = cruise_info('NAME','OVIDE 2002',...
'PI_NAME','H.Mercier','PI_ORGANISM','LPO-CNRS',...
'SHIP_NAME','Thalassa','SHIP_WMO_ID','FNFP',...
'DATE',[datenum(2002,6,18,0,0,0) datenum(2002,7,10,0,0,0)],'N_STATION',100);

Axes info variable geoinfo:

geoinfo.STATION_NUMBER = [1:100]';
geoinfo.STATION_DATE = linspace(datenum(2002,6,18,0,0,0),...
datenum(2002,7,10,0,0,0),100)';
geoinfo.LATITUDE = linspace(40.3,59.8,100)';
geoinfo.LONGITUDE = linspace(350.5,317.4,100)';
geoinfo.POSITIONING_SYSTEM = 'GPS';
geoinfo.PRES = meshgrid(1035*9.8*[0:10:5330]/10000,1:100);
geoinfo.MAX_PRESSURE = 1035*9.8*5330/10000*ones(100,1);
geoinfo.DEPH = meshgrid(0:-10:-5330,1:100);

Datas into variable dat:

t = odata('name','TEMP','long_name','Temperature','unit','degC','cont',...

meshgrid(linspace(2,1,100),1:534)'.*meshgrid(1+20*exp(geoinfo.DEPH(1,:)/5e2),1:100));

s = odata('name','PSAL','long_name','Salinity','unit','psu','cont',...

35.5*ones(100,534));

sl = odata('name','OXSL','long_name','Oxygen Solubility','unit','ml/l');
dat = struct('TEMP',t,'PSAL',s,'OXSL',sl);

Now we create the Transect object:

T = transect('source','The best lab in the world !','creator','John Doe',...

'file','no_source_file_at_this_point','file_date',datenum(1000,1,1,0,0,0),...
'cruise_info',C,'geo',geoinfo,'data',dat)

The empty variable OXSL (content of OData object sl was set to nothing) has a Real status and therefore is in the empty data list. Let's move it to a Virtual status:

>> T.data.PARAMETERS_STATUS(dstatus(T,'OXSL',1))='V'
[...]

4) ======================================== DATA LIST
                  1 (Virt): [OXSL] Oxygen Solubility
                  2 (Real): [PSAL] Salinity
                  3 (Real): [TEMP] Temperature
Try disp(T,2) for a more extensive description of datas for this transect object

It is now in the data list with the correct status. If we call it, the content will automatically be filled (we need temperature and salinity to compute the oxygen solubility, which are both available):
